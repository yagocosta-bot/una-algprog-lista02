Exercício 1 - Sensor de Ré  
Este exercício consiste em criar um fluxograma que represente o funcionamento básico de um sensor de ré automotivo. A lógica inicia com a leitura da distância até o obstáculo e, a partir disso, o sistema toma diferentes decisões:

Menos de 0,5 m: o carro está muito próximo, então o sensor deve emitir um bipe contínuo e exibir a mensagem "PARE".  
Entre 0,5 m e 2 m: o veículo está próximo, mas ainda com margem; assim, o sensor deve emitir um bipe intermitente.  
Acima de 2 m: não há risco imediato, então nenhum som é emitido.  

O objetivo é aplicar estruturas de decisão para representar cada uma dessas condições no fluxograma.  

Exercício 2 - Filtro do Instagram  

Neste exercício, o fluxo representa o processo simplificado de edição e publicação de uma foto. O usuário começa selecionando uma imagem, e o sistema o direciona para duas decisões consecutivas:  

Aplicar filtro preto e branco?  
Se sim, a imagem é convertida para tons de cinza.  
Se não, permanece como está.  

Postar agora?  
Se sim, a foto é enviada para o servidor.  
Se não, ela é salva localmente na galeria.  

O exercício destaca o uso de múltiplas decisões encadeadas, mostrando como diferentes escolhas do usuário afetam o resultado final.
